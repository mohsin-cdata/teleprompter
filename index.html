Setup Promptr

One-time Firebase setup — takes 5 minutes, free forever.

  1. Open console.firebase.google.com
  2. Create a new project (any name)
  3. Click Web to add a web app, copy the config values below
  4. Go to Build → Realtime Database → Create database → Test mode
  5. Paste values and save
